Js demo about ES5 & ES6 vanilla js functionalities.

Frontend development and Javascript have evolved a lot these last year. It became obvious to me that our backend oriented team have left behind a load of interesting improvements in Javascript. This demo is an attempt to illustrate some of the "new" functionalities that we have not yet adopted.

We'll be working with these files :

  • js/app.js -- main lab file
  • js/util.js -- collection of utilities, some already demonstrated code may be move there from app.js when checking out to a new step.
  • app/index.js -- this file, the one where we'll display the rendering of our code.

Go ahead and open this project in your IDE, then checkout to the first step of this demonstration.

git checkout 1/introduction
